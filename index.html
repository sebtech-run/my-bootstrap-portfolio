Présentation

Bonjour, je m'appelle Bouillet Sébastien. J'ai 48 ans et je suis actuellement en fin de parcours d'intégrateur web avec OpenClassrooms.
Passionné par l'informatique dès mon plus jeune âge, grâce en autres au site du zéro (nouvellement openclassrooms), j'ai d'abord travaillé dans le hardware en montage, réparation avant de ma passionner pour les langages informatiques. À ce jour, j'ai suivi une formation au Cnam de la Réunion de 1 année en tant que développeur web en apprenant des langages tels que PHP avec Laravel et en backend SqlServer, mySql.
Au fil de cet apprentissage ma préférence s'est tout naturellement dirigée vers le front-end, le design web. Cette formation OpenClassrooms m'a permis de me familiariser avec des langages front-end comme React ou encore Sass et Bootstrap. Mon objectif est d'acquérir de l'experience dans le développement web en travaillant sur des nouveaux projets.

ArgentBank

Dans ce projet, je travaille sur le développement front-end d'une application bancaire en utilisant React et Redux. Ma mission principale sera d'intégrer le front-end avec le back-end via des appels API. J'ai utilisé React pour développer l'interface utilisateur de l'application bancaire, en me concentrant sur la création d'un tableau de bord complet et responsive pour les utilisateurs.
Redux sera utilisé pour gérer les données de l'application. Cela nous permettra de maintenir un état global cohérent à travers l'application.
Pour définir les points d'accès de l'API et modéliser les interactions avec les données des transactions, j'ai utilisé Swagger. Cet outil me permet de décrire les différentes routes et actions nécessaires pour l'API. J'utilise Nodejs pour exécuter le code JavaScript côté serveur.

Objectifs compétences

Afficher les données du back end sur l'interface via des appels API

Configurer des routes API pour la communication client / serveur

Implémenter la gestion des données avec Redux pour assurer le fonctionnement du front

React Redux Swagger Api-Rest
